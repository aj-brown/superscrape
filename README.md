# Superscrape

A web scraping project using Camoufox and TypeScript.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Run the example in development mode:

```bash
npm run dev
```

## Build

Compile TypeScript to JavaScript:

```bash
npm run build
```

## Run

Run the compiled code:

```bash
npm start
```

## Project Structure

```
superscrape/
├── src/
│   └── index.ts       # Main entry point
├── dist/              # Compiled output (generated)
├── package.json
├── tsconfig.json
└── README.md
```

## About Camoufox

Camoufox is a privacy-focused browser automation tool built on Firefox that helps avoid detection.
